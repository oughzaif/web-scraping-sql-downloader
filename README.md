 Web Scraping SQL Downloader

This is a simple Python script that downloads a `.sql` file from a public legal URL using the `requests` library. It is designed for educational and cybersecurity research purposes.

 Features

- Uses the `requests` library to fetch data from a target URL.
- Saves the content as a local `.sql` file.
- Useful for beginners learning about web interaction and ethical scraping.

 Usage

1. Make sure you have Python installed.
2. Install the required library:

```bash
pip install requests
````

3. Run the script:

```bash
python downloader.py
```
